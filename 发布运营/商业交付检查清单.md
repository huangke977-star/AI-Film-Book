# 商业交付检查清单

## 1. 客户需求确认

- 项目目标是否明确。
- 发布平台是否明确。
- 时长和画幅是否明确。
- 品牌视觉限制是否明确。
- 是否涉及肖像、商标、版权素材。
- 是否需要配音、字幕、音乐。

## 2. 制作前交付物

- Brief。
- 脚本。
- 分镜表。
- 风格参考。
- 报价。
- 时间计划。

## 3. AI 生成阶段

- 每个镜头有独立提示词。
- 每个镜头至少有 2 个候选版本。
- 失败样本已记录。
- 重要角色或产品没有明显变形。
- 没有不可控商标、乱码文字、错误品牌元素。

## 4. 剪辑阶段

- 节奏符合平台。
- 字幕无错别字。
- 音乐和音效授权明确。
- 画面风格统一。
- Logo、产品、卖点出现时间符合客户要求。
- 配音、音乐和音效的来源已记录。
- 如使用声音克隆，已取得明确授权并记录使用范围。
- 如使用 AI 音乐，已记录工具、生成日期、账号状态和商用范围。
- 字幕文件或烧录字幕版本已经和客户确认。

## 5. 终审阶段

- 导出分辨率正确。
- 文件格式正确。
- 封面图正确。
- 发布文案已准备。
- 客户确认版本号。
- 修改意见已归档。

## 6. 交付文件

```text
final_video.mp4
cover.png
publish_copy.md
subtitle.srt
voice_license.md
music_license.md
asset_sources.md
project_files.zip
cost_log.md
revision_log.md
```

## 7. 声音与音乐授权检查

| 检查项 | 说明 |
| --- | --- |
| 配音工具授权 | 当前账号是否允许该项目用途 |
| 声音克隆授权 | 是否有本人或权利方明确同意 |
| 音乐工具授权 | AI 音乐平台当前条款是否允许该用途 |
| 上传素材权利 | 上传到 AI 音乐工具的音频是否有权使用 |
| 客户复用范围 | 客户是否可二次剪辑、跨平台发布或长期投放 |
| 撤回机制 | 声音或音乐授权出现争议时如何替换 |
