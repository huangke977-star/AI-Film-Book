# 电影感风格模板

## 基础风格

```text
真实电影摄影，低饱和，冷暖对比，浅景深，柔和侧逆光，轻微胶片颗粒，自然肤色，稳定镜头，真实空间层次。
```

## 可调参数

### 色彩

- 冷色科技感：蓝灰、青色、低饱和。
- 暖色生活感：钨丝灯、暖黄、柔和阴影。
- 商业广告感：干净、明亮、高质感。
- 悬疑感：低照度、高反差、局部光。

### 光线

- 侧逆光：适合人物和产品轮廓。
- 窗边自然光：适合生活方式和访谈。
- 屏幕光：适合创作者、程序员、AI 工作流主题。
- 霓虹光：适合未来城市、科技主题。

### 镜头质感

- 浅景深。
- 真实镜头畸变。
- 轻微胶片颗粒。
- 自然动态模糊。
- 不要过度锐化。

## 负面约束

```text
不要卡通，不要塑料质感，不要过度磨皮，不要廉价广告感，不要乱码文字，不要多余手指，不要多余人物，不要夸张表情。
```
