# 故障排查

## 人物变脸

可能原因：

- 没有角色圣经。
- 每个镜头的人物描述不同。
- 运镜过于复杂。
- 镜头时长过长。

解决方法：

- 固定人物描述。
- 使用参考图。
- 减少环绕和大幅度运动。
- 把长镜头拆成多个短镜头。

## 手部错误

可能原因：

- 手部动作过细。
- 要求敲键盘、拿杯子、指向屏幕等复杂动作。

解决方法：

- 改成“手停在键盘附近轻微移动”。
- 避免手部特写。
- 用剪辑和音效暗示动作。

## 屏幕文字乱码

可能原因：

- 模型不擅长直接生成可读文字。

解决方法：

- 提示词中写“抽象界面”。
- 不要求生成具体中文。
- 后期用剪辑软件添加真实文字。

## 镜头运动失控

可能原因：

- 同时要求多种运镜。
- 使用“dynamic”“fast”“dramatic”等过强词。

解决方法：

- 每个镜头只写一种运动。
- 加入“slow, stable, subtle”。
- 必要时改成固定镜头。

## 风格不统一

可能原因：

- 没有统一 style guide。
- 每个镜头临时写提示词。
- 光线、色彩、场景不断变化。

解决方法：

- 建立 `style_guide.md`。
- 每个提示词都引用相同风格段落。
- 把色彩、光线、质感固定下来。
