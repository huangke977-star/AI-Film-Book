# 2026 AI 视频工具清单

本清单用于长期维护，不做一次性排名。工具变化很快，价格、模型和功能应定期核验。

## 1. 对话与 Agent

| 工具 | 主要用途 | 适合阶段 |
| --- | --- | --- |
| ChatGPT | 创意、解释、改写、脚本讨论 | 选题、脚本、复盘 |
| Codex | 文件管理、项目协作、批量生成文档和提示词 | 全流程 |
| MCP | 让 Agent 连接外部工具 | 自动化 |
| n8n | 工作流自动化 | 批量任务 |

## 2. 视频生成

| 工具 | 主要用途 | 备注 |
| --- | --- | --- |
| LibTV | AI 视频素材生成、Agent 工作流接入 | 本项目重点工具 |
| Seedance | 视频生成模型能力，适合放入文生视频、图生视频和多镜头复盘工作流中理解 | 第 06 卷重点讲解，具体入口和权限以官方页面及实际平台为准 |
| Veo | 高质量视频生成 | 注意地区和权限 |
| Kling | 生视频、图生视频 | 适合短视频创作者 |
| PixVerse | 快速生视频 | 适合测试 |
| Hailuo | 生视频 | 适合中文用户 |
| Pika | 创意视频生成 | 适合特效感 |

## 与第 06 卷的关系

第 06 卷把 Seedance 或同类视频模型放进生成工作流中理解。本清单只做工具定位，不替代当前官方页面、实际账号权限和平台价格核验。

## 3. 图像生成

| 工具 | 主要用途 |
| --- | --- |
| GPT Image | 概念图、参考图、封面 |
| FLUX | 写实图像和风格图 |
| Ideogram | 图文设计 |
| Recraft | 商业视觉、图标、插画 |

## 4. 声音和音乐

| 工具 | 主要用途 | 备注 |
| --- | --- | --- |
| OpenAI Audio / TTS | 文字转语音、音频生成和语音工作流方向 | 具体模型、音色、费用和 API 字段以官方文档为准 |
| ElevenLabs | 英文配音、声音克隆、品牌声音方向 | 声音克隆必须确认授权、本人验证、账号权益和商用范围 |
| Fish Audio | 声音克隆和配音 | 中文生态常见选择之一，商用规则需按当前官方页面确认 |
| 豆包 | 中文配音 | 适合中文短视频场景，具体入口和权限以实际平台为准 |
| MiniMax Speech | 中文语音 | 可作为中文 TTS 方向关注，具体能力以官方页面为准 |
| CosyVoice | 开源/本地语音方向 | 适合技术用户了解本地语音方向，新手先不必深挖 |
| Suno | AI 音乐生成 | 商用、版权和分发规则需按官方 Terms / Rights / Ownership 页面确认 |
| Udio | AI 音乐生成 | 使用上传音频前必须确认输入素材权利和平台条款 |

## 与第 11-12 卷的关系

第 11 卷把配音当作信息时间轴处理，第 12 卷把音乐当作情绪时间轴处理。本清单只做工具定位，不替代官方条款、账号权益和商业授权核验。

## 5. 剪辑和后期

| 工具 | 主要用途 | 备注 |
| --- | --- | --- |
| 剪映 / CapCut | 中文短视频剪辑、字幕、模板、快速导出 | 适合新手先完成短视频闭环，不写死按钮路径 |
| Premiere Pro | 商业剪辑流程、字幕、导出和团队工作流 | 第 13 卷用于说明专业剪辑流程，具体功能以 Adobe 官方帮助为准 |
| DaVinci Resolve | 调色、Fairlight 音频和专业后期 | 适合后续深入调色与声音后期 |
| Final Cut Pro | Mac 剪辑流程 | 适合 Mac 用户，高级能力按 Apple 官方文档确认 |

## 与第 13 卷的关系

第 13 卷不要求新手同时掌握所有后期软件。短视频可先用剪映完成闭环，商业和团队项目再逐步学习 Premiere Pro；DaVinci Resolve 和 Final Cut Pro 作为后续专业方向。

## 6. 评估维度

后续排名不只看“画面好不好”，还要看：

- 一致性。
- 可控性。
- 生成速度。
- 成本。
- API/Agent 接入能力。
- 商用授权。
- 中文生态。
- 失败重试成本。
