# 2026 AI 视频制作完全指南：从零到商业变现

这是一个可长期维护的 AI 视频制作百科项目，目标不是写一份普通教程，而是沉淀成一套可以持续更新、可以导出 PDF/EPUB、可以作为创作者工作台使用的 Markdown 电子书与资源库。

当前资料包基于交接说明和案例线索整理而成。Ross 抖音案例的原始短链为 <https://v.douyin.com/PUBrROriSnE/>，当前环境未能直接抓取视频画面和完整字幕，因此案例拆解以用户给出的标题信息、Codex 工作方式、LibTV 技能公开说明和 AI 视频制作通用流程为基础，所有推断会在文档中标明。

## 推荐阅读顺序

1. 先看 `项目交接说明.md`，确认整体目标。
2. 再看 `学习路线.md`，选择适合自己的路线。
3. 如果目标是复刻 Ross 那类“Agent + AI 视频平台”的短视频，直接从 `案例拆解/Ross_Codex_LibTV_抖音案例.md` 和 `工作流/Codex_LibTV_端到端视频制作流程.md` 开始。
4. 开始实操时，把每次作品新建为一个独立项目，复用 `Prompt库/` 和 `分镜模板/` 里的结构。

## 项目原则

- 用项目文件管理创作，而不是只在聊天框里临时写提示词。
- 把“提示词”升级为“上下文工程”：角色设定、镜头表、风格约束、资产清单、迭代记录都要落到文件。
- 先追求稳定可复用，再追求炫技。
- 所有案例拆解都区分事实、可验证来源和推断，方便后续补充真实截图、视频帧和操作记录。
