# 2026 AI 视频制作完全指南：从零到商业变现

这是一个可长期维护的 AI 视频制作百科项目，目标不是写一份普通教程，而是沉淀成一套可以持续更新、可以导出 PDF/EPUB、可以作为创作者工作台使用的 Markdown 电子书与资源库。

当前资料包基于交接说明和 Ross 抖音案例整理而成。Ross 抖音案例的原始短链为 <https://v.douyin.com/PUBrROriSnE/>，已于 2026-07-10 打开核验到真实视频页、标题、作者、发布时间、时长、章节要点和关键画面。案例拆解会区分页面可验证信息、画面观察和基于工作流的制作推断。

## 当前推荐学习入口

第一阶段请先学第 01-05 卷，再进入 14 天训练营。第二阶段请在第 01-05 卷之后学习第 06-10 卷。它的重点是视频生成模型、镜头语言、导演判断、人物一致性和 AI 绘图参考图。不要从第 11-18 卷开始，因为这些卷仍属于后续阶段。

推荐顺序：

```text
第01卷 -> 第02卷 -> 第03卷 -> 第04卷 -> 第05卷 -> 14天入门训练营 -> Ross 案例复盘
第06卷 -> 第07卷 -> 第08卷 -> 第09卷 -> 第10卷
```

## 推荐阅读顺序

1. 先看 `学习路线.md`，理解为什么先学第 01-05 卷。
2. 按顺序读 `第01卷_AI视频基础.md` 到 `第05卷_LibTV完全教程.md`。
3. 再进入 `14天入门训练营.md`，每天只完成一个小输出。
4. 阅读 `案例拆解/Ross_Codex_LibTV_抖音案例.md` 和 `工作流/Codex_LibTV_端到端视频制作流程.md`。
5. 按顺序读 `第06卷_Seedance完全教程.md` 到 `第10卷_AI绘图.md`，建立画面生成和导演判断能力。
6. 开始实操时，把每次作品新建为一个独立项目，复用 `Prompt库/` 和 `分镜模板/` 里的结构。

## 项目原则

- 用项目文件管理创作，而不是只在聊天框里临时写提示词。
- 把“提示词”升级为“上下文工程”：角色设定、镜头表、风格约束、资产清单、迭代记录都要落到文件。
- 先追求稳定可复用，再追求炫技。
- 所有案例拆解都区分事实、可验证来源和推断，方便后续补充真实截图、视频帧和操作记录。
