# Prompt 库

这里保存可复用提示词。原则是：提示词不是孤立句子，而是项目上下文的一部分。

## 分类建议

- `选题生成`
- `脚本生成`
- `分镜生成`
- `角色一致性`
- `风格统一`
- `LibTV生图`
- `LibTV生视频`
- `剪辑复盘`
- `发布文案`

## 标准结构

每条 Prompt 建议包含：

```markdown
# Prompt 名称

## 使用场景

## 输入文件

## 提示词正文

## 输出格式

## 注意事项

## 示例
```

## 使用原则

- 能引用文件就不要重复粘贴大段背景。
- 每次只解决一个明确任务。
- 让 Codex 先列缺失信息，再开始生成。
- 保存成功版本，也保存失败版本。
